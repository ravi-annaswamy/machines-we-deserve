# The Machines We Deserve

An illustrated longread article weaving five AI news stories from March 12, 2026 into a literary magazine-style narrative about risks, possibilities, and societal change.

**Built entirely in one session using Claude (Anthropic) in Cowork mode:**

1. Browsed Google News → Technology → Artificial Intelligence using Chrome automation
2. Captured and categorized 30+ headlines into thematic clusters
3. Selected 5 stories forming a compelling narrative arc
4. Researched each story in depth via web search
5. Wrote a 3,000-word New Yorker-style longread
6. Generated 5 pen-and-ink cartoon illustrations via Google Gemini
7. Assembled into a polished, responsive HTML page

## The Five Stories

- **The Oppenheimer Problem** — Anthropic's standoff with the Pentagon over AI weapons restrictions
- **The Agents That Learned to Conspire** — AI agents autonomously collaborating on cyberattacks
- **The Workers Behind the Curtain** — Kenyan data labelers organizing for recognition
- **The Quiet Apocalypse at Oracle** — Mass layoffs explicitly credited to AI efficiency
- **The Bubble Question** — Silicon Valley's growing unease about AI valuations
